# Demo Scripts

This directory contains demonstration scripts for testing and showcasing features.

## ⚠️ Important Notes

- **Demo scripts are for development/testing only**
- These scripts create test data and may modify the database
- **Never run demo scripts in production**
- Demo scripts are for educational/development purposes only

## Available Demos

### `userManagementDemo.js`
Demonstrates user management features including:
- User creation
- Profile updates
- Address management
- Preferences management
- User statistics
- Search functionality
- Activity tracking
- Referral system

## Usage

To run a demo:
```bash
node demos/userManagementDemo.js
```

## Security Warning

⚠️ **Demo scripts create test data with default credentials.**
- Change all passwords after running demos
- Remove or secure demo scripts before production deployment
- Never commit demo scripts with real user data
